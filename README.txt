== Description ==
Script-Style-Killer Is A Nice Little Plugin: It Uses A Ruthless HTML-Manipulation At A Level Beyond WordPress-API, To Remove Scripts And Styles From The HTML, By Commenting Them-Out (With A Reason Also Detailed), ModPageSpeed Can Remove The Comments, Minifing The Page Overall-Text-Length.

== Installation ==
1.  Upload to folder to the <code>/wp-content/plugins/</code> directory.
2.  Optional- Edit the <code>main.php</code> to your own-values or just let it be.
3.  Activate the plugin ('Plugins' menu), No configuration needed.
4.  enjoy.
